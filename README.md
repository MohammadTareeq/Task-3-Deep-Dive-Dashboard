# Task 3 – Deep-Dive Analysis & Interactive Dashboarding

## Internship
ApexPlanet Software Pvt. Ltd. – Data Analytics Internship

## Objective
To define key business KPIs, perform a focused deep-dive analysis, and build an interactive dashboard using Google Sheets.

## KPIs Defined
- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Discount Bands

## Deep-Dive Analysis
The analysis focused on understanding the impact of discount levels on profitability by categorizing discounts into bands and analyzing profit behavior.

## Dashboard
An interactive dashboard was built using pivot tables and slicers to dynamically filter and explore insights.

## Tools Used
